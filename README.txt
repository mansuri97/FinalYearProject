README

How to run the web application:
1) Download XAMPP, which is a server that allows you to run PHP and the web application as a whole
2) Next, find the folder called htdocs, in this folder there should be another folder called 'example'
3) Move all of my code into this 'example' folder
4) On the XAMPP control panel, press the start button for both 'Apache' and 'MySQL'
5) go to local host
6) register a new user account, and log in
